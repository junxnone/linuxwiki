# Steps
1. 下载最新 `Release`
2. 获取 `Github token`
3. 产生 `hash` 值
> hash 和主域名绑定的，当主域名更新后，需要更新hash值
4. 修改 `index.html`
5. 上传到仓库
6. 设置repo为github pages
