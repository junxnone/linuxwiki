# 实用工具

Command | Description
-- | --
du -h | 统计文件夹占用空间大小
tree | 显示文件树
tmux | 后台终端工具
meld | 文件比较工具 GUI
xargs | find . -name *.txt \| xargs cat 
pv | pv  file1 > file1.copy (显示copy进度)
cal | 打印日历
time | `time ./your_app` 计算执行时间
