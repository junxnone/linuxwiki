# Brief
- **top** 查看系统进程资源占用


# UseCase

Usecase | cmd
-- | --
CPU 使用率排序 | <kbd>Shift</kbd> + <kbd>P</kbd>
Memory 使用率排序 | <kbd>Shift</kbd> + <kbd>M</kbd>
查看使用的 CPU Core | <kbd>f</kbd> <br> 选中选项 ` P       = Last Used Cpu (SMP)`
